# 📊 Financial Causality Analysis using Knowledge Graphs

## 📌 Overview
This project builds a financial knowledge graph to analyse cause-and-effect relationships from company 10-K filings. The goal is to extract meaningful insights from unstructured financial text and identify key drivers affecting business performance.

---

## ❓ Problem Statement
Financial reports (10-K filings) contain large volumes of unstructured text, making it difficult to identify relationships between key business factors such as revenue, costs, and risk drivers.

---

## 💡 Solution
Developed an NLP-based pipeline to:
- Extract cause-effect relationships from financial text
- Structure them into a knowledge graph
- Analyse trends across companies and time periods

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- NLP: SpaCy, BERT
- Graph Database: Neo4j
- Visualisation: Matplotlib

---

## ⚙️ Approach
1. Collected and preprocessed 10-K filings
2. Applied NLP techniques to extract causal relationships
3. Structured extracted data into nodes and relationships
4. Stored data in Neo4j knowledge graph
5. Analysed and visualised key patterns

---

## 📈 Key Insights
- Identified recurring financial drivers impacting company performance
- Highlighted relationships between risk factors and revenue trends
- Enabled structured exploration of financial cause-effect patterns

---

## 📊 Output Examples
- Graph visualisation (Neo4j)
- Relationship networks
- Key trend plots

## 🚀 Business Value

- Enables structured analysis of unstructured reports
- Can be extended for risk analysis and investment insights
